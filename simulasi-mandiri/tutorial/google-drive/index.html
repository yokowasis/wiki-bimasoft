# Setting Google Drive untuk format jawaban Upload

# Pembuatna Auth Token

  1. Silakan generate token Google Drive melalui Link Auth

  2. Masukkan token yang digenerate pada webadmin > pengaturan test > Google Drive Token

# Pembuatan Folder Penampungan Jawaban Siswa

  1. Buat Folder pada Google Drive sekolah untuk menampung jawaban siswa

  2. Copy ID Folder yang sudah sudah di buat ke dalam Webadmin > Pengaturan test > Gogle Drive Parent
