# Unblock / Mengenable Macro Pada Template Word / Excel

Karena alasan security terkadang Windows dan Microsoft Office melakukan pemblokiran terhadap file yang di download dari internet, termasuk file Template Word / Excel. Oleh karena itu kita perlu membuka blokir tersebut.

  1. File -> Option

  1. Trust Center -> Trust Center Settings...

  1. Macro Settings -> Enbal VBA

  1. Close Microsoft Excel
  2. Explorer / File Manager > Klik Kanan Template Excelnya > Properties

# Khusus Untuk Microsoft Office 365 / Office 2020 ke atas.

  1. Centang Unblock
